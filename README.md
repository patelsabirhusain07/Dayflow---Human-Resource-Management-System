
# Human Resource Management System (HRMS)

##  Overview
The **Human Resource Management System (HRMS)** is a centralized web-based application designed to automate and manage core HR activities such as employee records, attendance, leave, payroll, and performance management. The system improves efficiency, accuracy, and transparency within an organization.

This project is developed as part of a **hackathon** to solve real-world HR management challenges.

---

##  Problem Statement
Many organizations still rely on manual processes or disconnected tools like spreadsheets to manage HR operations. These methods are time-consuming, error-prone, and lack transparency, making it difficult to track employee data and make informed decisions.

---

##  Solution
The HRMS provides a single integrated platform where HR managers and employees can securely access and manage HR-related information. By automating routine HR tasks, the system reduces manual effort, improves data accuracy, and enhances organizational productivity.

---

##  Features
- Employee profile and records management  
- Attendance tracking  
- Leave application and approval  
- Payroll management  
- Role-based access control (Admin / HR / Employee)  
- Secure authentication  
- User-friendly dashboard
